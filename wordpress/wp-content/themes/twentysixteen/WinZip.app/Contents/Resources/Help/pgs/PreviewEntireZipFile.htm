Quick Look (Previewing an entire Zip file)

WinZip® expands the Finder's Quick Look by adding support for previewing the content of your Zip file. Select a Zip file in the Finder and press the Space bar or click the Quick Look icon in the Finder window. A transparent window appears, showing you the contents of the Zip file instantly. The Quick Look window for a Zip file looks like the following:

  1. Open with WinZip button: click on Open with WinZip to lanuch WinZip and open the previewed Zip file.
  2. Summary Bar: provides summary information about the Zip file as a whole. Summary information includes the Zip file name, the kind of file (same as what appears in the Kind column in Finder), Zip file size, and the date the Zip file was last modified.
  3. File/Folder List: displays a list with detailed information about the files and folders in the Zip file. The list behaves like List view mode of the WinZip main window. For each file, details include whether or not the file is encrypted, the file's icon, the file's name, the date the file was last modified, the uncompressed and compressed size of the file, and the kind of file. For each folder, the details include the last modification date of the folder. Folders are normally closed. To expand the folder and view its content, click the triangle to the left of the folder icon.
