Emailing the link of an Open Zip File

To email a link to a WinZip® file stored in the Cloud, create or open a Zip file. Click on the Email the link icon on the toolbar or click on Email the link in the Actions menu and select the cloud service to be used. If the selected cloud service is not already logged in, a cloud service login dialog will be presented. Enter your account information and follow the cloud service login instructions.

The Zip file will be uploaded to the Cloud and the link to the Zip file will be inserted into a new email message. Complete the message and send it out.

  1. Zip File Link: this is a link to the Zip file that has been saved in your Cloud Storage Service.

To email the link of files from within WinZip®, first add them to an open Zip file and then email the Zip file. To email the link of the open Zip file, click on the Email the link icon on the toolbar or click on Email the link from the Actions menu.
