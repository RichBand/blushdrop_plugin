Burning an Open Zip File to Disc

To burn files from within WinZip®, first add them to an open Zip file and then burn the Zip file. To burn the open Zip file, click on Burn to Disc from the Actions menu. The following sheet will display:



  1. Drive Selection: If there are multiple drives that can burn discs, select the drive to be used to burn the Zip file to a disc.
  2. Progress Meter: Once the burn operation has been started, the meter will indicate the current progress of burning the Zip file onto the disc.
  3. Burn: After selecting the drive to use for burning, click the Burn button to start burning the open Zip file to the disc. If there is not a blank disc in the selected drive, this button is disabled.

When the burn operation has completed, the sheet will change and display Burn Complete:



Click on Close and the burn operation is complete.
