The Help Menu

The Help menu provides access to information about using WinZip®.

  1. Search: search the content of this help file for information using words or phrases you type.
  2. WinZip Help: displays the content topic of this help file. From the content topic, you can find and navigate to information about various features provided by WinZip.
  3. Information on the Web: up to date information and answers to frequently asked questions can be found in the WinZip knowledge base on the web. You can send Feedback about WinZip to WinZip Computing. If you have a problem, you can also contact Technical Support.
