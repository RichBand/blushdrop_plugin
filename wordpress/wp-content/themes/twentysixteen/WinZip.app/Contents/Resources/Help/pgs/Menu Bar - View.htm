The View Menu

The View menu lets you specify how the Zip file content is presented and arranged in WinZip®

  1. Views and Bars: the upper section of the View menu allows you to specify how your files and folders are displayed in the WinZip main window. Like Finder, your choices are as Icons, as List, as Columns, as Cover Flow, and as Flat. The next section, Arrange By,lets you pick how the files and folders are ordered in the WinZip main window. The last section allows you to hide or show the various bars that make up the WinZip interface. The bars you can show or hide are the Zip bar, the Status bar, and the Toolbar.
  2. Arrange by Choices: the Arrange By feature lets you select how the files and folders are ordered in the WinZip main window. To select an ordering, choose one of the columns listed in the Arrange By menu. Only columns that are shown can be selected. Selecting a column that is already selected reverses the order.
