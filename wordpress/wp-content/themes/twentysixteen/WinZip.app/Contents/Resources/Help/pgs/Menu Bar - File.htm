The File menu

The File menu provides WinZip®'s file and folder operations on the Zip file or selected files in the Zip file.

  1. Zip File/Folder features
    • New Zip file creates a new Zip, Zipx, or LHA file (depending on your selected default format in the WinZip Preferences) in a new WinZip window.
    • New Folder creates a new folder in the current Zip file. Files and Folders can then be dragged and dropped into the new folder
    • Open Zip File displays a list of possible locations from where to open a Zip file.
      • from this Mac displays a dialog box where you can locate and select a Zip file from this Mac and then open it in WinZip.
      • from iCloud Drive displays a dialog box where you can locate and select a Zip file from iCloud Drive and then open it in WinZip (this option is only available on OS X 10.10 or later).
      • from ZipShare displays a dialog box where you can locate and select a Zip file from ZipShare and then open it in WinZip. If not logged into ZipShare, a login dialog is shown first.
      • from Dropbox displays a dialog box where you can locate and select a Zip file from Dropbox and then open it in WinZip. If not logged into Dropbox, a login dialog is shown first.
      • from Google Drive displays a dialog box where you can locate and select a Zip file from Google Drive and then open it in WinZip. If not logged into Google Drive, a login dialog is shown first.
    • Open Recent opens a previously opened Zip file from the provided list. The list can also be cleared so previously opened Zip files are no longer listed.
  2. Individual File features
    • Close closes the current Zip file without exiting WinZip.
    • Save displays a list of possible locations where a Zip file can be saved.
      • to this Mac displays a dialog box where you can save your current Zip file to another location.
      • to iCloud Drive displays a dialog box where you can save your current Zip file to select location in iCloud Drive (this option is only available on OS X 10.10 or later).
      • to ZipShare displays a dialog box where you can save your current Zip file to ZipShare. If not logged into ZipShare, a login dialog is shown first.
      • to Dropbox displays a dialog box where you can save your current Zip file to a location in Dropbox. If not logged into Dropbox, a login dialog is shown first.
      • to Google Drive displays a dialog box where you can save your current Zip file to a location in Google Drive. If not logged into Google Drive, a login dialog is shown first.
    • Rename Selected File renames the selected file in the Zip file.
    • Move Selected File(s) To moves the selected file(s) in the Zip file to another folder in the same Zip file.
    • Delete Selected File(s) delete the selected file(s) from the Zip file. This operation cannot be undone. The operation does not delete the original files from which the Zipped versions were created and it does not delete the Zip file.
  3. Open Enclosing Folder
    • Opens Finder showing the content of the folder where the open Zip file is located.
  4. Open a file in a Zip file
    • Get Info displays information on the selected file such as the file's name and kind, modification date, and zipped and unzipped sizes.
    • Quick Look "filename" displays the selected file's content in Quick Look.
    • Open "filename" opens the file using the associated program.
  5. Opening a file in a Zip file
    • Open With opens the selected file using the program you choose from the list of associated programs. If the file(s) are changed, you will be prompted to update the Zip file when the WinZip window is reactivated.
