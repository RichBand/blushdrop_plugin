Automatically Create an Empty Zip File

In order to easily create Zip files with WinZip®, a new "Untitled.zip" file will be created when WinZip starts up.

Creating an Empty Zip File

To create a new Zip file at any time while WinZip is running, click on New Zip File from the File menu.


The new, empty Zip file is created and an open Zip file window displays. You can now use Add to select files and folders to be zipped and added to the Zip file.
