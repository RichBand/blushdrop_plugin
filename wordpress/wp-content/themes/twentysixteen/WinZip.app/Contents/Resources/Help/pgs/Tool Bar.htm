The Toolbar

The WinZip® window contains a toolbar with the normal controls for closing, minimizing, and enlarging the WinZip window as well as the name of the Zip file being displayed in the window. In addition, the toolbar contains buttons similar to the toolbar in Finder and buttons specific to WinZip. The toolbar for WinZip appears as follows:

The toolbar provides quick and easy access to frequently used operations.

  1. View: click on the Icon, List, Column, Cover Flow or Flat button to change the WinZip display to the corresponding view. The behavior matches that of Finder.
  2. Add: click on Add to drop down a menu of add "from ..." locations. Select files/folders and then zip, optionally encrypt, and add them to the Zip file.
  3. Unzip: click on Unzip to drop down a menu of unzip "to ..." locations. Unzip the entire content or selected files/folders in the Zip file to a specified folder or cloud storage.
  4. Email: click Email as attachment or Email the Link to create a new email message in Apple Mail with the Zip file as an attachment or a file download link.
  5. Share: click on Share to upload the Zip file to a cloud service and then select a share service (Mail, Messages, Twitter, Facebook, LinkedIn or more).
