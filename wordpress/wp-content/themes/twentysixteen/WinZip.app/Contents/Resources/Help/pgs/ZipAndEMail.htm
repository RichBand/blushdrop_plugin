Creating a New Zip File and Emailing

To create a Zip file and Email it using WinZip®, use one of the following options:
  • Zip and Email using the Dock: drag and drop files/folders on the WinZip icon on the Dock. Select the Zip and Email tab to select the Zip and Email pane. Follow the Zip and Email instructions provided in Using the Dock Icon.
  • Email as Zip File from the right-click shortcut menu (or CTRL + click if not using a mouse with a right mouse button): in a Finder window, select the files/folders that you wish to email as a Zip file. Right-click on the selected files/folders (or CTRL + click) and select Email as Zip File from the bottom of the main right-click shortcut menu or from the Services submenu item if that is present. On the Zip and Email dialog that opens, make any desired changes to the options and click Compose Message to zip the selected files and attach the new Zip file to a new email message.
