The Menu Bar

When WinZip® is the active program, the Mac menu bar changes to the following:

The Menu bar contains new and modified menus that provide access to the features of WinZip. The menus offer the following groups of features:

  • The WinZip menu is where to go to: get information about, purchase, register, receive updates and notices, and configure WinZip.
  • The File menu is where to go for file and folder operations on the Zip file or selected files in the Zip file such as new or open.
  • The Edit menu is where to go for standard editing functions such as copy and paste.
  • The View menu is where to go to configure how the Zip file content is presented and arranged such as icons or a list arranged by name or kind.
  • The Go menu is where to go for folder navigation operations such as back and forward.
  • The Actions menu is where to go for zipping and unzipping features such as add or unzip.
  • The Window menu is where to go to manage your WinZip windows such as size and order.
  • The Help menu is where to go to get information about using WinZip.
