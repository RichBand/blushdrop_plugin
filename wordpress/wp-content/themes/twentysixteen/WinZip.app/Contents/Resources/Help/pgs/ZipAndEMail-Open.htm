Emailing an Open Zip File

To email files from within WinZip®, first add them to an open Zip file and then email the Zip file. To email the open Zip file, click on the Email as attachment icon on the toolbar or click on Email as attachment from the Actions menu. The following sheet will display:



  1. Picture Size: If the Zip or Zipx file contains images, this sheet appears when you click on the Email icon or menu item. The sheet offers the option to resize pictures in the Zip file attached to the message. When the Zip file includes pictures, additional space savings can be obtained by reducing the picture size. Using this feature does not alter the size of the originals in the Zip file. Only pictures stored in a temporary copy of the Zip file to be attached to the email message are resized. Pictures are only reduced, never enlarged. The aspect ratio, the ratio of the width to the height of the picture, is also maintained. The choices for size are Actual Size (the image size is not changed), Smaller (the picture is reduced to fit in an area 640 x 480 pixels), Small (the picture is reduced to fit in an area 800 x 600 pixels), Medium (the picture is reduced to fit in an area 1024 x 768 pixels), and Large (the picture is reduced to fit in an area 1280 x 1024 pixels).

    2. Once the desired option is selected, click on Compose Message.

The Zip file will be attached to a new message. Simply, finish the message and send it.
