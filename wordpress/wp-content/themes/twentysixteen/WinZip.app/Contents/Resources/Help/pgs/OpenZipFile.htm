Opening an existing Zip file

In order to zip and unzip files and folders with WinZip®, you need to have a Zip file. You can use an existing Zip file or you can create a new, empty Zip file. To open an existing Zip file, choose one of the following methods
  • drop a Zip file on the WinZip icon on the Dock.
  • click on Open Zip File from the File menu to drop down a menu of locations.
  • click on Open Recent from the File menu to drop down a list of recently opened Zip files.
Using Open Zip File and selecting a location will display the standard File Open dialog box. Navigate to the folder containing the desired Zip file, select it, and click Open.
