Technical Support

We think you will find WinZip® easy to use. The answers to most questions can be found in this help file. Additional information about WinZip may be available from the WinZip web site.

If these do not resolve your problem, technical support is available at no charge as described below. The best way to report problems is with the support form available from the WinZip web site. You can also send email to the address shown on the WinZip web site's contact page, or postal mail to WinZip Computing, P.O. Box 540, Mansfield, CT 06268.

When reporting problems, please include the following information:

  1. Is the problem reproducible? If so, how?
  2. What version of Mac are you using: Mavericks? Yosemite?
  3. What version of WinZip are you running? Please include the entire version number e.g., "4.0.2515".
  4. If a dialog box with an error message was displayed, please include the FULL TEXT of the dialog box, including the text in the title bar.
