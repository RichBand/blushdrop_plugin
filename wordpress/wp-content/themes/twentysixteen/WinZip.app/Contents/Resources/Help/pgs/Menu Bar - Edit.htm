The Edit Menu

The Edit menu provides standard editing functions for the Zip file using WinZip®.

The edit operations include Cut, Copy, Paste, Select All and Special Characters.

When a Zip file is open, Select All selects all the files in the Zip file so that they can be unzipped, deleted, etc.

Cut, Copy, Paste, and Select All can be used in edit fields. The Special Characters feature allows you to insert special characters in edit fields.
