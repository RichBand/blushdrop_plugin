The Go Menu

The Go menu provides operations for navigating through folders in a Zip file opened in WinZip®.

  1. Navigation: Back and Forward navigates through the folders as if they were web pages, taking you back to a previously viewed folder or ahead to a folder from which you just navigated back. Enclosing Folder takes you to the folder that contains the folder you are viewing. This may not be the same folder from which you came and the result may differ from selecting Back.
