Using Drag and Drop

Zipping and unzipping files in WinZip® is as easy as dragging and dropping files.
Zipping Using Drag and Drop
Zipping files using Drag and Drop can be accomplished in one of two ways
  • Using the Dock Icon: From Finder or the desktop, select the files to be zipped, drag and drop them on the WinZip icon on the Dock. See Using the Dock Icon for more details.
  • Using an Open Zip file: from an open Zip file navigate to the location where the zipped files are to be added. From Finder or the desktop, select the files to be zipped, drag and drop them inside the WinZip window. The files will be zipped using the format that corresponds to the type of Zip file on which the files were dropped. Files cannot be encrypted using this method, use the WinZip icon on the Dock to encrypt the files.
Unzipping Using Drag and Drop
To unzip files using drag and drop, open the Zip file in WinZip. Locate the files to be unzipped. Drag and Drop the files from WinZip onto the desktop or into a folder in a Finder window. If the files are encrypted, you may be prompted for the password.
