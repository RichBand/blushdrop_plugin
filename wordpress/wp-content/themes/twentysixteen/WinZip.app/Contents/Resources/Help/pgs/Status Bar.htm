Status Bar

The status bar appears at the bottom of the WinZip® window. It provides information about the Zip file and items selected in the Zip file. When there are no items selected in the Zip file, the status bar appears like this:


Zip File Information: the status bar displays the number of files/folders in the Zip file, the total space needed on disk when the files/folders are unzipped, and the total space the files/folders are using in their zipped form.



When items are selected in the Zip file, the status bar looks like this:


Selected File Information: the information for selected files indicate how many of the total files/folders in the Zip file are selected, the space the files/folders will use on the disk if unzipped, and the amount of space the zipped files/folders use.
