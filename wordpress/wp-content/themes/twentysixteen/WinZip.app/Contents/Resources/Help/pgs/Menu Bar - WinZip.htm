The WinZip menu

When WinZip® appears as the first menu on the Menu bar, it indicates WinZip is the active program. Go to this menu to get information about, purchase, register, receive updates and notices, and configure WinZip.

  1. WinZip Menu Items
    • About WinZip displays a standard About dialog box. Information includes, the WinZip copyright information, version number and the user's registration information.
    • Check for updates performs a check to see if a new version of WinZip is available or if there is any other information from WinZip.
    • Buy Now is displayed when evaluating WinZip. This option displays a dialog box where WinZip can be purchased.
    • Register is displayed when evaluating WinZip. This option displays a dialog box where you can enter your registration or activation information to register your copy of WinZip.
    • Uninstall uninstalls WinZip from the Mac, removing the application.
    • Preferences displays the Preferences dialog box where the settings for WinZip can be viewed and changed.
  2. Mac Menu items
    • The remaining items on the menu are standard application menu items. They behave like the same functions as found in other programs' application menus
