Saving a Zip file

To save a Zip file from within WinZip®, you need to have a Zip file open. To save a Zip file, click on Save from the File menu and select a location from the drop down menu. The following sheet will display:
Save to this Mac or to iCloud Drive

  1. Save Sheet: when the to this Mac menu item is selected, the Save sheet slides down from the WinZip toolbar. You can then select the folder location where to save the open Zip file. Once the selection has been made, click Save to save the file to the selected folder.
  2. Save As File Name: the top of the Save sheet contains a textbox to type the file name to save the Zip file. The name shows "Untitled" because that is the default name of new Zip files created.
  3. Folder Selection: the upper half of the Save sheet contains the standard Mac Save As dialog box fields, allowing navigation through devices, places, and folders and the selection of the folder to save the Zip file.
  4. New Folder: the bottom left of the Save sheet contains the New Folder button to create a new folder under the current folder.
Save to a Cloud service

Select a cloud storage service from the list of available services in WinZip.


  1. Save Sheet: When the Save to <Cloud Service> menu item is selected, the Save sheet slides down from the WinZip toolbar. You can then select the folder location to save the open Zip file to the selected cloud storage service. Once the selection has been made, click Save to save the file to the selected location.
  2. Save As File Name: The top of the Save sheet contains a textbox to type the file name for the Zip file.
  3. Folder Selection: The upper half of the Save sheet contains a Mac style-like dialog box allowing navigation through the cloud folders and the selection of the folder to save the Zip file.
  4. New Folder: The bottom left of the Save sheet contains the New Folder button to create a new folder under the current folder in the cloud storage.
