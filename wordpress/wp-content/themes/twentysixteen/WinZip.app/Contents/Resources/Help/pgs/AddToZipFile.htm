Adding files to an open Zip file (Zipping)

To zip files from within WinZip®, add them to an open Zip file. A Zip file is open if you created the Zip file or opened the Zip file using WinZip. To add files to the open Zip file, click on the Add from My Mac or Add from Cloud icons on the toolbar or click on Add from My Mac or Add from Cloud from the Actions menu. The following sheet will display:
Add from My Mac

  1. Add Sheet: When the Add from My Mac icon or menu item is selected, the Add sheet slides down from the WinZip toolbar. You can then select files and folders to be added to the open Zip file. Once the selection has been made, click Zip to add the files to the Open Zip file
  2. File/Folder Selection: The upper half of the Add sheet contains the standard Mac Open dialog box fields allowing navigation through devices, places, and folders and the selection of files and folders.
  3. Add Options: The lower half of the Add sheet contains the Add option for encrypting the files being zipped. Check the Encrypt box and enter the password and re-enter the password for spelling verification. The encryption method used is based on the setting in the Preferences.
Add from Cloud

Selection a cloud storage services from the list of available services in WinZip.


  1. Add Sheet: When the Add from Cloud icon or menu item is selected, the Add sheet slides down from the WinZip toolbar. You can then select files and folders to be added to the open Zip file. Once the selection has been made, click Zip to add the files to the Open Zip file
  2. File/Folder Selection: The middle of the Add sheet contains the file/folder list allowing selection of files and folders.
  3. Add Options: The lower half of the Add sheet contains the Add option for encrypting the files being zipped. Check the Encrypt box and enter the password and re-enter the password for spelling verification. The encryption method used is based on the setting in the Preferences.

Note: the type of compression used when zipping files depends on the kind of Zip file to which the files are being added. For example, files with a .zip extension use Zip (compatibility) while files with a .zipx extension use Zipx (best compression).
